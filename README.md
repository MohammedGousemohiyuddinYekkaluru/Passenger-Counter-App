# Passenger Counter App

A simple web application to track and display the count of passengers in real time.

## Features

- Increment and decrement the count.  
- Display current count with basic styling.  
- Built using HTML, CSS and JavaScript.

## Technologies Used

- HTML  
- CSS  
- JavaScript

## Usage

1. Clone the repository:  
   ```bash
   git clone https://github.com/MohammedGousemohiyuddinYekkaluru/Passenger-Counter-App.git
   ```
2. Open index.html in your browser.

3. Use the “INCREMENT” and “DECREMENT” buttons to change the count.

4. The current count will be displayed on screen.
